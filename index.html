Anagram

Two word are said to be anagram of each other if they have same set of character in them ...or we can say they are made up of same word justi by rearanging the character only.For example,rat and art made of same character just differ in arrangement..other examples are raaz and azra.

Logic

To check two string are anagram of each other or not you have to make two array containing corresponding frequencies of each character this can be done by iterating each string for each character and noting fre quencies of each charcter .Then we have to compare two array if they are eqaul then they are anagram of each other else,Not.
